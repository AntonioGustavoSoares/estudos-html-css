Links Externos

Para coloca Links externo basta usa a tag Âncora <a> e </a>. Como por exemplo o Youtube ← Clique Aqui para acessa.

Link Externos em outra Janela

Youtube Shorts ← Clique aqui para assistir vídeos curtos. Lembrando que para fazer isso você deve colocar o atributo "target", caso queira o site em outros idiomas basta colocar o atributo "hreflang" e assim indicar qual é o idioma que deseja.

Links Internos

Para colocar links internos é basicamente a mesma coisa, use a tag de Âncora <a> e </a>. Mas as únicas diferenças estão dentro dos atributos, lembrando, que o uso de links internos são mais utilizados para acessar outra página do seu próprio site.
