Principais formatações

Negrito/Destaque:

Como sabemos a tag <b> </b> é uma tag OBSOLETA, por isso, usaremos a tag semântica <strong>

Exemplo:
Nessa frase estamos usando a tag <b>. O rei reivindicou o trono.

Nessa frase estamos utilizando a tag <strong>. A madame escorregou no chão.

Itálico/Ênfase:

Já a tag antiga do itálico ou ênfase era a tag <i> pois também se tornou OBSOLETA, por isso, atualmente é usado a tag sêmantica <em>.

Exemplo:
Nessa frase usaremos a tag <i>. O cachorro ficou muito contente em ver o dono.

Desta vez usaremos a tag <em>. Depois que a poeira se decipou vimos o caminho para a montanha
